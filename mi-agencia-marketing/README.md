**The Data Rebels** es una plataforma web de alto impacto diseñada para una agencia de marketing de vanguardia. El proyecto rompe con la estética corporativa tradicional para adoptar un estilo **Cyberpunk / Tech-Noir**, enfocado en la transparencia de datos y el ROI exponencial.

---

## 🚀 Concepto de Negocio
Nuestra agencia no hace marketing convencional; hackeamos el mercado. Utilizamos:
* **Análisis Masivo de Datos:** Para encontrar patrones donde otros ven caos.
* **Estrategia de Guerrilla:** Posicionamiento disruptivo basado en métricas reales.
* **Sistemas Autónomos:** Implementación de IA para optimizar conversiones 24/7.

## 🛠️ Stack Tecnológico
* **Frontend:** React.js (Componentización funcional).
* **Bundler:** Vite (Velocidad de desarrollo optimizada).
* **Estilos:** Tailwind CSS (Arquitectura basada en utilidades y Responsive Design).
* **Iconografía:** Emojis y tipografía monoespaciada para un look de terminal.



## ✨ Características Principales
* **Modo Oscuro/Claro Dinámico:** Cambio de interfaz en tiempo real mediante el estado de React.
* **Hero Section con Video:** Capas de glassmorphism sobre video en loop para una experiencia inmersiva.
* **Servicios Dinámicos:** Mapeo de datos (`.map()`) para facilitar la escalabilidad de los planes de la empresa.
* **Arquitectura "Slate-950" en Footer:** Un pie de página estilizado en azul medianoche profundo para separar visualmente la jerarquía del sitio.
* **Diseño Responsive:** Adaptabilidad total desde dispositivos móviles hasta pantallas Ultra-Wide.

## 🎨 Sistema de Diseño
* **Color Primario:** `#39FF14` (Neon Green) - Representa energía, código y acción.
* **Color de Fondo:** `#050505` (Cyber Black) - Profundidad técnica.
* **Acento:** `Slate-950` - Elegancia y profesionalismo en el cierre de página.
* **Tipografía:** Combinación de Sans-Black (Impacto) y Monoespaciada (Tecnología).
